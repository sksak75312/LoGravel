# Tailwind CSS 完全指南：從入門到精通

Tailwind CSS 是一個 utility-first 的 CSS 框架，它提供了大量的低層級工具類別，讓你能夠快速建構自訂設計而不需要離開 HTML。這篇文章將帶你從基礎概念開始，逐步掌握 Tailwind CSS 的各種功能和最佳實踐。

---

## 📌 什麼是 Tailwind CSS？

Tailwind CSS 採用「utility-first」哲學，提供數千個預建的 CSS 類別，每個類別都有單一的功能。相比傳統的 CSS 框架（如 Bootstrap），Tailwind 讓你能夠直接在 HTML 中組合這些工具類別來建立獨特的設計。

### 傳統 CSS vs Tailwind CSS

```html
<!-- 傳統 CSS 方式 -->
<div class="card">
  <h2 class="card-title">標題</h2>
  <p class="card-content">內容文字</p>
</div>

<style>
.card {
  background-color: white;
  border-radius: 8px;
  padding: 24px;
  box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
}

.card-title {
  font-size: 1.5rem;
  font-weight: 600;
  margin-bottom: 16px;
}
</style>

<!-- Tailwind CSS 方式 -->
<div class="bg-white rounded-lg p-6 shadow-md">
  <h2 class="text-2xl font-semibold mb-4">標題</h2>
  <p class="text-gray-600">內容文字</p>
</div>
```

## 🎨 核心概念：Utility Classes

### 間距系統 (Spacing)

Tailwind 使用一致的間距尺度，基於 `0.25rem` (4px) 的倍數：

```html
<!-- Padding -->
<div class="p-4">全方向 padding: 16px</div>
<div class="px-6 py-3">水平 24px，垂直 12px</div>
<div class="pt-8 pb-4 pl-2 pr-6">各方向獨立設定</div>

<!-- Margin -->
<div class="m-4">全方向 margin: 16px</div>
<div class="mx-auto">水平置中</div>
<div class="mt-8 mb-4">上下不同間距</div>

<!-- 負值 margin -->
<div class="-mt-4">負 margin</div>
```

### 顏色系統

Tailwind 提供了豐富的顏色調色盤：

```html
<!-- 背景顏色 -->
<div class="bg-blue-500">藍色背景</div>
<div class="bg-gray-100">淺灰背景</div>
<div class="bg-red-600">深紅背景</div>

<!-- 文字顏色 -->
<p class="text-green-700">綠色文字</p>
<p class="text-purple-400">紫色文字</p>

<!-- 邊框顏色 -->
<div class="border border-yellow-300">黃色邊框</div>

<!-- 陰影顏色 -->
<div class="shadow-lg shadow-blue-500/50">藍色陰影</div>
```

### 字體與文字

```html
<!-- 字體大小 -->
<h1 class="text-6xl">超大標題</h1>
<h2 class="text-4xl">大標題</h2>
<p class="text-base">一般文字</p>
<small class="text-sm">小字</small>

<!-- 字重 -->
<p class="font-thin">極細</p>
<p class="font-normal">正常</p>
<p class="font-semibold">半粗</p>
<p class="font-bold">粗體</p>

<!-- 文字對齊 -->
<p class="text-left">靠左</p>
<p class="text-center">置中</p>
<p class="text-right">靠右</p>
<p class="text-justify">兩端對齊</p>

<!-- 行高與字距 -->
<p class="leading-tight tracking-wide">緊密行高，寬字距</p>
```

## 📱 響應式設計

Tailwind 採用 mobile-first 的響應式設計方法：

```html
<!-- 響應式斷點：sm (640px), md (768px), lg (1024px), xl (1280px), 2xl (1536px) -->
<div class="w-full md:w-1/2 lg:w-1/3">
  <!-- 手機全寬，平板半寬，桌面三分之一寬 -->
</div>

<!-- 響應式文字大小 -->
<h1 class="text-2xl md:text-4xl lg:text-6xl">
  響應式標題
</h1>

<!-- 響應式 flexbox -->
<div class="flex flex-col md:flex-row">
  <div class="w-full md:w-2/3">主內容</div>
  <div class="w-full md:w-1/3">側邊欄</div>
</div>

<!-- 響應式顯示/隱藏 -->
<div class="hidden md:block">桌面顯示</div>
<div class="md:hidden">手機顯示</div>
```

### 實用的響應式元件範例

```html
<!-- 響應式導覽列 -->
<nav class="bg-white shadow-lg">
  <div class="max-w-7xl mx-auto px-4">
    <div class="flex justify-between items-center py-4">
      <div class="text-xl font-bold">Logo</div>
      
      <!-- 桌面選單 -->
      <div class="hidden md:flex space-x-8">
        <a href="#" class="text-gray-700 hover:text-blue-600">首頁</a>
        <a href="#" class="text-gray-700 hover:text-blue-600">關於</a>
        <a href="#" class="text-gray-700 hover:text-blue-600">服務</a>
        <a href="#" class="text-gray-700 hover:text-blue-600">聯絡</a>
      </div>
      
      <!-- 手機選單按鈕 -->
      <button class="md:hidden">
        <svg class="w-6 h-6" fill="none" stroke="currentColor">
          <path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M4 6h16M4 12h16M4 18h16" />
        </svg>
      </button>
    </div>
  </div>
</nav>
```

## ⚙️ 自訂配置

### tailwind.config.js 設定

```javascript
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    "./src/**/*.{html,js,tsx,jsx}",
    "./components/**/*.{html,js,tsx,jsx}",
  ],
  theme: {
    extend: {
      // 自訂顏色
      colors: {
        'brand': {
          50: '#eff6ff',
          500: '#3b82f6',
          900: '#1e3a8a',
        },
        'custom-gray': '#f8f9fa',
      },
      
      // 自訂字體
      fontFamily: {
        'sans': ['Inter', 'system-ui', 'sans-serif'],
        'heading': ['Poppins', 'sans-serif'],
      },
      
      // 自訂間距
      spacing: {
        '18': '4.5rem',
        '88': '22rem',
      },
      
      // 自訂斷點
      screens: {
        'xs': '475px',
        '3xl': '1600px',
      },
      
      // 自訂動畫
      animation: {
        'fade-in': 'fadeIn 0.5s ease-in-out',
        'slide-up': 'slideUp 0.3s ease-out',
      },
      
      keyframes: {
        fadeIn: {
          '0%': { opacity: '0' },
          '100%': { opacity: '1' },
        },
        slideUp: {
          '0%': { transform: 'translateY(10px)', opacity: '0' },
          '100%': { transform: 'translateY(0)', opacity: '1' },
        },
      },
    },
  },
  plugins: [
    require('@tailwindcss/forms'),
    require('@tailwindcss/typography'),
    require('@tailwindcss/aspect-ratio'),
  ],
}
```

## 🚀 效能優化

### 移除未使用的 CSS

```javascript
// tailwind.config.js
module.exports = {
  content: [
    './src/**/*.{html,js,jsx,ts,tsx}',
  ],
  // Tailwind 會自動移除未使用的類別
}
```

### 使用 JIT 模式

JIT (Just-In-Time) 模式讓 Tailwind 只生成你實際使用的樣式：

```javascript
// tailwind.config.js
module.exports = {
  mode: 'jit', // 現在預設啟用
  content: ['./src/**/*.{html,js}'],
  // ...
}
```

### 優化建議

1. **使用語義化的元件類別**：將常用的樣式組合封裝成可重複使用的類別
2. **善用 CSS 變數**：對於主題色彩等經常變動的值
3. **響應式設計優先**：從手機版開始設計
4. **保持一致性**：建立設計系統和樣式指南
