## 前言

鏈結列表(Linked List)是一種線性的資料結構，但是以非連續性的方式儲存資料，則是透過每個節點去指向下一個節點的位置，每個節點基本會有兩個參數，分別為節點的值及下一個節點的位置。

非連續性的資料儲存，就是每筆資料儲存在記憶中的不同位置，想的簡單一點就是各自分散在記憶體中，所以才需要透過節點去指向之後的節點位置。而有非連續性儲存資料就會有連續性的儲存資料，在程式中最常碰到的也最簡單的連續性儲存資料就是陣列(Array)，Array 會透過 index 的索引值 0,1,2,3...等來完成連續性儲存的特性。它們兩者也是最容易被拿來比較的，下面再來深入說明兩者的差別。

```tsx
class ListNode<T> {
  constructor(
    public val: T,
    public next: ListNode<T> | null = null,
  ) {}
}

const n1 = new ListNode(1);
const n2 = new ListNode(2);
const n3 = new ListNode(3);
const n4 = new ListNode(4);

n1.next = n2;
n2.next = n3;
```
